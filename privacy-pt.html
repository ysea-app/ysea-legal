POLÍTICA DE PRIVACIDADE

Esta Política de Privacidade para Setty PON (a operar sob o nome comercial Ysea) («nós» ou «nosso») descreve como e por que razão podemos aceder, recolher, armazenar, utilizar e/ou partilhar («tratar») as suas informações pessoais quando utiliza os nossos serviços («Serviços»).

Dúvidas ou questões? Contacte-nos em yseaapp_services@outlook.com.

1. RESPONSÁVEL PELO TRATAMENTO

Setty PON, a operar sob o nome comercial Ysea
E-mail: yseaapp_services@outlook.com

2. QUE DADOS RECOLHEMOS

Dados da conta

  • Endereço de e-mail e palavra-passe (com hash pela Supabase Auth)
  • Foto de perfil (opcional)

Perfis de crianças

  • Nome, género e data de nascimento da criança
  • Data prevista de parto (perfis de gravidez)
  • Foto de perfil da criança (opcional)

Fotos faciais diárias

A aplicação oferece uma funcionalidade opcional de foto facial diária para acompanhar o crescimento da criança ao longo do tempo. Estas fotos são armazenadas nos nossos servidores (Supabase, região UE) e nunca são partilhadas com terceiros. Pode eliminar fotos individuais em qualquer momento a partir da aplicação.

Entradas no diário

  • Textos, fotos, gravações de áudio e vídeos que adiciona
  • Dados de localização GPS (opcional): Quando ativado, as suas coordenadas geográficas são associadas às entradas do diário para recordar onde uma memória teve lugar. Estes dados nunca são utilizados para fins publicitários ou de rastreamento comportamental.
  • Etiquetas definidas pelo utilizador

Utilização do assistente de IA

  • Mensagens enviadas ao assistente
  • Contador diário de utilização de mensagens

Dados de subscrição

As compras são geridas pelo RevenueCat e pelas lojas de aplicações (App Store / Google Play). Não armazenamos os dados do seu cartão de pagamento.

3. COMO UTILIZAMOS OS SEUS DADOS

  • Para fornecer e operar os Serviços (conta, perfis, diário, assistente)
  • Para personalizar as respostas da IA com o contexto do seu filho
  • Para gerir subscrições e quotas de armazenamento
  • Para enviar notificações administrativas (segurança, atualizações de política)
  • Para garantir a segurança e prevenir fraudes

4. PARTILHA DE DADOS E TRANSFERÊNCIAS INTERNACIONAIS

Não vendemos as suas informações pessoais. Trabalhamos com os seguintes subcontratantes:

Supabase (base de dados e autenticação)

Todos os seus dados são armazenados na Supabase, alojada na União Europeia (região West EU Irlanda). A Supabase está sujeita às regulamentações europeias de proteção de dados.

RevenueCat (gestão de subscrições)

O RevenueCat processa informações de subscrição (ID de utilizador, nível, histórico de compras). Sede nos Estados Unidos. A transferência é regida pelas Cláusulas Contratuais Padrão (SCC) aprovadas pela Comissão Europeia.

Anthropic (assistente de IA) — Transferência fora da UE

Quando utiliza o assistente de IA, o nome, a idade, o género e o perfil do seu filho, juntamente com a sua mensagem, são transmitidos à Anthropic PBC (Estados Unidos) para gerar a resposta.

Esta transferência é regida pelas Cláusulas Contratuais Padrão (SCC) aprovadas pela Comissão Europeia (Decisão 2021/914). A Anthropic não utiliza os seus dados para treinar os seus modelos e não conserva estes dados para além do processamento do pedido.

Política de privacidade da Anthropic: anthropic.com/privacy

5. CONSERVAÇÃO DE DADOS

  • Dados ativos: conservados enquanto a sua conta estiver ativa.
  • Após eliminação da conta: todos os seus dados (perfis, diário, fotos, gravações) são eliminados no prazo de 30 dias.
  • Dados de subscrição: conservados pelo RevenueCat em conformidade com as suas obrigações legais e fiscais.
  • Registos de segurança: conservados por um máximo de 90 dias por razões de segurança.

6. OS SEUS DIREITOS (RGPD)

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD), tem os seguintes direitos:

  • Direito de acesso (art. 15.º): obter uma cópia dos seus dados.
  • Direito de retificação (art. 16.º): corrigir dados inexatos.
  • Direito ao apagamento (art. 17.º): eliminar os seus dados — diretamente a partir de Definições → Eliminar a minha conta.
  • Direito à portabilidade (art. 20.º): receber os seus dados num formato estruturado — a partir de Definições → Exportar os meus dados.
  • Direito de oposição (art. 21.º): opor-se ao tratamento com base em interesses legítimos.
  • Direito de limitação (art. 18.º): solicitar a suspensão temporária do tratamento.
  • Direito de retirar o consentimento em qualquer momento, sem afetar a licitude do tratamento anterior.

Para exercer estes direitos: yseaapp_services@outlook.com

Tem também o direito de apresentar uma reclamação junto da autoridade de controlo local (em França: CNIL).

7. SEGURANÇA

Implementamos medidas técnicas e organizativas adequadas para proteger os seus dados: encriptação TLS em trânsito, isolamento de dados por utilizador (Row Level Security), tokens de acesso aleatórios para ficheiros armazenados, chaves de API apenas do lado do servidor.

Nenhum sistema é completamente seguro. Em caso de violação de dados que o afete, notificá-lo-emos em conformidade com o artigo 34.º do RGPD.

8. DADOS DE CRIANÇAS

A Ysea foi concebida para pais e tutores legais. A aplicação recolhe dados sobre crianças (perfil, fotos, marcos de desenvolvimento) por instrução do progenitor que cria a conta. Estes dados nunca são utilizados para fins comerciais ou publicitários.

Se acredita que foram recolhidos dados sobre uma criança sem o seu consentimento, contacte-nos imediatamente.

9. COOKIES

A aplicação móvel Ysea não utiliza cookies. Os dados são armazenados localmente no seu dispositivo (AsyncStorage) exclusivamente para a funcionalidade da aplicação (sessão, preferências de idioma, notificações).

10. ATUALIZAÇÕES DA POLÍTICA

Podemos atualizar esta política periodicamente. Para alterações significativas, notificá-lo-emos através da aplicação. A data da última atualização é indicada no topo deste documento.
